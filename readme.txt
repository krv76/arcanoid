
Основа для тестового задания.

Проект под Microsoft Visual C++ 6.0

Менять надо файл game.cpp, можно добавить свои модули и т.д.

Цель: не более чем за 4 часа сделать простой арканоид: управляемая
ракетка, мячик, который летает, отскакивает от стенок и ракетки и
разбивает кирпичи. Игра должна как-то реагировать на падение мячика
вниз, например рестартовать или заканчиваться.

Прочие навороты (при наличии времени), вроде падающих бонусов
приветствуются.

Рекомендуется сделать так, чтобы мячиков в будущем можно было сделать
несколько.

Вырисовку делать попиксельно в буфер (32-битный цвет BGRA) - см. game.cpp.
Достаточно (но необязательно) изображать всё (включая мячик) прямоугольниками.
